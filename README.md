# curso-dev-tabnews

Implementação do [TabNews](https://www.tabnews.com.br/) feito no curso [curso.dev](https://curso.dev)

### rodar projeto

```bash
npm run dev
```

### rodar test

```bash
npm test
npm test:watch
```

### rodar docker

```bash
docker compose up -d
```

### finalizar docker

```bash
docker compose down
```

### rodar psql

```bash
psql --host=localhost --username=postgres --port=5432
```

### rodar prettier

```bash
npm run lint:check
npm run lint:fix
```
